Чтобы Employee мог на портале видеть и редактировать кастомную таблицу
надо

1\) В Manager в OneIM Manager – Base Role создать Application role с
Perm Group = CCCEditRole

2\) Поместить Employee в эту App Role

Не работает для сотрудника, к которому привязана административная
системна уз 1IM
