1.  Stop debugging.

> Terminate batch jobs: In every open terminal press Ctrl+C, reply Y to
> question Terminate batch job (Y/N)?

2.  In terminal window run command:

> **npm run build:app qer-app-portal (** **npm run build:lib qer)**
>
>       The result will appear in the \imxweb\dist folder
>
>  

3.  Open folder \imxweb\dist\qer-app-portal.

>       Delete folder \html ???
>
>       Using 7-Zip, zip content of the folder and give it a
> name “Html_qer-app-portal.zip”
>
>  

4.  Move zip file to the
    folder **C:\inetpub\wwwroot\ApiServer\bin\imxweb\custom**

>       Original file can remain in
> the C:\inetpub\wwwroot\ApiServer\bin\imxweb folder
>
>  

5.  Run iisreset

>
