25 сентября 2025 г.

11:48

 

1.  Установка утилит для разработки (через chocolatey):

- VS Code ([Visual Studio Code - Code Editing.
  Redefined](https://code.visualstudio.com/))

> choco install vscode

- NodeJS ( <https://nodejs.org/en/>)

> choco install nodejs-lts --version=16.15.0 (20.19.4)

- Git ( <https://git-scm.com>)

> choco install git -y

- Postman ( <https://www.postman.com>)

> choco install postman -y

2.  Настройка VSCode. Установка плагинов

- Angular Language Service

- NX console

- HTML Css Support

- CSS Formatter

3.  Установить API Server

4.  Клонировать репозиторий

> git clone --branch v92
> <https://github.com/OneIdentity/IdentityManager.Imx.git>
> ./identitymanager-imx-vid

5.  Установить зависимости Angular проекта

> cd identitymanager.imx-vid/imxweb/
>
> npm install

6.  В
    \identitymanager-imx-vid\imxweb\projects\qer-app-portal\src\environments\environment.ts
    указать адрес до API сервера

7.  Запуск сборки

> *QBM -\> QER -\> qer-app-portal*
>
>  
>
> npm run build:watch qbm
>
> npm run build:watch qer
>
>  
>
> *\# --\> Start the Web Portal*
>
> npm run start qer-app-portal
