## Title

> SSO (Single Sign-On) is not working for the Web Portal

## Description

> SSO (Single Sign-On) is not working. What are the required steps to
> configure this?

## Resolution

> Single Sign-On is possible for the One Identity (1IM) Web Portal (IT
> Shop) if the following conditions are met:  
>   
> - The configuration parameter
> **TargetSystem\ADS\AuthenticationDomains** contains a valid value,
> e.g. domain.com.  
>   
> In **Designer** select **Getting Started** then **Edit configuration
> parameters:  
>   **
>  
>
> <img src="../media/21. One Identity SSO Portal/media/image1.png"
> style="width:8.33333in;height:5.09375in" />
>
> Click To See Full Image.
>
> \- The appropriate Authentication Module, i.e. **Active Directory user
> account (role based)**, is enabled.  
>   
> In **Designer** select **Base Data**, expand **Security settings** and
> select **Authentication modules**:  
>   
>  
>
> <img src="../media/21. One Identity SSO Portal/media/image2.png"
> style="width:8.4375in;height:2.73958in" />
>
> Click To See Full Image.
>
> \- **Anonymous Authentication** for the web site is disabled in IIS:  
>   
>  
>
> <img src="../media/21. One Identity SSO Portal/media/image3.png"
> style="width:5in;height:1.90625in" />
>
> Click To See Full Image.
>
> \- The logged in user, e.g. logged in to a Windows system, has a valid
> ADSAccount and role-based Employee in the 1IM database, e.g.:  
>   
>  
>
> <img src="../media/21. One Identity SSO Portal/media/image4.png"
> style="width:4.84375in;height:5in" />
>
> Click To See Full Image.
>
> \- It may be necessary to update Internet Explorer to use the
> **Automatic logon with current username and password** option.  
>   
>  
>
> <img src="../media/21. One Identity SSO Portal/media/image5.png"
> style="width:3.78125in;height:5in" />
>
> Click To See Full Image.
>
> \- It is assumed the 1IM web portal is configured correctly and the
> appropriate Authentication module is selected in the web.config.  
>   
>  
>
> <img src="../media/21. One Identity SSO Portal/media/image6.png"
> style="width:5in;height:4.45833in" />
>
> Click To See Full Image.
>
>
