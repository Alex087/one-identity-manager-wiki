19 декабря 2022 г.

11:45

В браузере найти нужный класс

В WebDesigner поиском найти отвечающий элемент за этот класс CSS

Делаем копию элемента без замены

Делаем правку класса в новом элементе

В компоненте WebDesigner указываем наш новый CSS элемент

Скрыть кол-во найденных записей в Grid

.imx-grid-paging-totalcount { display: none; }

.imx-nodataindication { display: none; }
